# Python-Mini-Projects
A Collection of Python Mini-Projects built to strength the core concepts of python programming , problem solving-skills, APIs and Algorithmic Thinking.

#Projects Included
1. Number Guessing Game
- Console-based game
- Uses loops, conditionals, random module
- Focus: control flow and user input handling

2. Rock Paper Scissors
- User vs Computer game
- Tracks scores
- Focus: game logic and loops

3. Timed Math Challenge
- Math quiz with time tracking
- Calculates total time and wrong attempts
- Focus: time module and performance tracking

4. Password Manager
- Stores encrypted passwords
- Uses file handling and encryption
- Focus: security basics and file operations

5. Slot Machine Game
- Simulates slot machine betting
- Uses weighted randomness
- Focus: probability, functions, and modular code

6. Turtle Racing Game
- Graphical turtle race simulation
- Random movement and winner detection
- Focus: Python turtle graphics

7. Typing Speed Test
- Terminal-based typing test
- Calculates WPM
- Focus: real-time input handling (curses module)

8. Maze Path Finder
- Visual pathfinding using BFS algorithm
- Uses curses for terminal visualization
- Focus: data structures, BFS, algorithmic thinking

9. Currency Converter (API Project)
- Fetches live currency rates using FreeCurrencyAPI
- Converts between currencies
- Focus: REST APIs, JSON parsing, error handling

Purpose of This Repository
- Strengthen Python fundamentals
- Practice real-world problem solving
- Prepare for advanced projects in AI / Data Science
- Build a public learning portfolio

#Note
These are learning projects, not production applications.  
Code quality and structure improve progressively across projects.
